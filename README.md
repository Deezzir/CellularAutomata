## Controls

| Key                                                         | Descritption                   |
|-------------------------------------------------------------|--------------------------------|
| <kbd>SPACE</kbd>                                            | Pause                          |
| <kbd>c</kbd>                                                | Clear board                    |
| <kbd>r</kbd>                                                | Randomize board                |
| <kbd>q</kbd>,<kbd>ESC</kbd>                                 | Quit                           |
